Newb Shader Help Downloads

Customization

of Newb X Legacy

To customize the shader, you would have to download the shader source code and setup the build environment.

If you are having issues following this guide, you can ask for help on the Discord server.

Android

Setup

Requires around 200Mb data to download necessary apps and files.

Termux
NMM File Manager

1. Install Termux (For building shader)

2. Install NMM File Manager (For editing) - You can also use Acode instead

3. Open Termux and run the following commands

pkg install openjdk-17 git --no-install-recommends
git clone https://github.com/devendrn/newb-x-mcbe
cd newb-x-mcbe
./setup.sh

4. Launch NMM File Manager and open sidebar. Click + icon and choose External Storage. Select Termux app in sidebar and accept permission.

Building

In Termux, go inside newb-x-mcbe folder

cd newb-x-mcbe

Now build files by:

./build.sh

To build only terrain shader (saves time):

./build.sh -m RenderChunk

The output will be in build folder.

To edit config, Open NMM, goto termux's home/newb-x-mcbe/include/newb_config_legacy.h. After making a change, build the shader and test it to make sure it works.

Windows

Setup

newb-x-mcbe-source.zip

1. Download shader repository zip

2. Extract contents into a folder

3. Go inside the folder and open cmd there

4. Run .\setup.bat

Building

To build all shader files, do:

.\build.bat

When editing config, you generally want to save build time by only building shader files you need:

.\build.bat -m RenderChunk

This will only build the terrain file. The output will be in build folder.

Config is inside include\newb_config_legacy.h. Open this file using a text editor to edit the values. After making a change, build the shader and test it to make sure it works.
