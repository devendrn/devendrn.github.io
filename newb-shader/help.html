Installation

of Newb X Legacy
*video tutorial uses different shaders as an example

Android

Using Patch apk (64-bit device only!)

*Due to obvious reasons, I cannot directly link the Patched apk here. You have to download it from the Discord server.

Patched Minecraft apk

1. Download and install the patched Minecraft apk that supports shaders.

2. Download the shader and import it into Minecraft.

3. Enable the shader in Global Resource Packs.

4. Now restart Minecraft.

Windows

Using BetterRenderDragon

BetterRenderDragon.zip

1. Download BetterRenderDragon.zip and extract the files.

2. Download the shader and import it into Minecraft.

2. Open Minecraft using LaunchMinecraft.bat of BetterRenderDragon. You have to launch Minecraft this way every time you want to use shaders.

3. Press F6 to open BetterRenderDragon window and enable the MaterialBin Loader option.

3. Enable the shader in Global Resource Packs and restart Minecraft.

FAQs

Frequently asked questions

Why is the patch apk not getting installed?

You are probably on a 32-bit device. The patch apk is only for 64-bit devices.

Can I use shaders in the normal Minecraft apk?

No, you cannot use shaders in normal Minecraft.

Why is the sky cutting off at the bottom?

You have to enable the shader in global resources and restart Minecraft to reload the cubemap.

Can I run other shaders on the patched apk?

Yes, the patch apk supports all RenderDragon shaders. Newb Shader is just one of them. See RenderDragon Shaders.

How can I customize the shader?

You can customize the shader, but it requires a little bit of hassle. See Shader customization.
